# si5351_Signal_Source
A very simple tool that generates a pulsing RF signal with pushbuttons to place it where you need it. 
Intended to test receivers, RF mixers, or drive exciters/transmitter PA chains.
Targets Arduino Nano/ATMega328, or XIAO ESP32-Cx 
Supports either a 128x32 OLED I2C display, or a 16x2 LCD.

In its current form, generates a pulsed carrier with ratio of on-to-off set in a #define

For a CW beacon see my other projects.
